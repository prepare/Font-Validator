Message: E2503

VDMX: The VDMX data can't be validated due to an error while rasterizing the glyphs, likely a hinting instruction error

Metrics affected by hints can only be validated at runtime. Therefore, this error is typically generated when the Font Validator uses the rasterizer to execute the hinting instructions contained in the font. The detail of this error message may or may not refer to the glyph that is associated with the bad hinting code, depending on how the overflow or other rasterizing error was encountered. For example, in the case of a stack overflow, the problem may be due to hinting instructions for a previous glyph not having been properly removed from the stack. To perform detailed checks of hinting instructions, select rasterization testing in the Font Validator, or use a tool such as Microsoft's Flint or VTT.

For more information, see the VDMX table specification.
