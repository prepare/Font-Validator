Message: E1121

glyf: Duplicated components of composite glyph

This composite glyph references a component more than once (or two different components with the same shape) and the duplicated component overlaps itself. The problem may have been caused by accidentally pasting a component twice while creating or editing the glyph. Remove the duplicated component using an outline editing tool. Remember to first identify which component is controlled via hints so that hinting information is not lost when the duplicate component is removed.

For more information, see the glyf table specification.
