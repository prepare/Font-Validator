Message: I1111

Application: Unable to perform test due to previously detected errors

A previously encountered error (likely in the glyf table) is preventing the Font Validator from performing the indicated test. For example, if there is a self-intersecting contour in the glyph, the test for misoriented contours may fail. Fix the previously encountered error using the appropriate tool and revalidate the font.

For more information, see the glyf table specification.
