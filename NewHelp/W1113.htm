Message: W1113

glyf: Degenerated contour(s)

This glyph contains a contour that is defined as one or more points in the same location. This type of contour will produce no shape when rendered by the rasterizer.

Contours defined this way are not illegal. They may be used to specify a component attachment point outside the base glyph.

For more information, see the glyf table specification.
