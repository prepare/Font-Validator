Message: E1117

glyf: Shift of component either undefined or ambigously defined

There is contradictory or insufficient information in the bitfield that defines the component shift. The placement of a component is specified in the glyf table as either: (1) x and y offsets, or (2) as two point numbers. To interpret the placement values, the rasterizer relies on the settings of bits 0 and 1 in the component flag. Bits 0 and 1 should be set as follows:

Bit 0 - If this is set, the placement values are words; otherwise they are bytes.
Bit 1 - If this is set, the placement values are xy offsets (in FUnits); otherwise they are point numbers.

For more information, see the glyf table specification.
